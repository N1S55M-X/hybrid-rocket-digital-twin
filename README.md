# 🚀 RocketPy Hybrid Digital Twin

## Overview
This project demonstrates a **Hybrid Digital Twin prototype** for rocket stabilization.  
It integrates:
- **RocketPy** for high‑fidelity flight dynamics and physics modeling  
- **PID controller** for thrust vector control (TVC)  
- **Neural Network (Keras)** to provide real‑time correction factors  
- **Matplotlib animation** for dynamic visualization  

The result is a hybrid simulation that blends physics‑based modeling with AI correction, illustrating how digital twin concepts can be applied to aerospace systems.

---

## Features
- Realistic rocket motor and aerodynamic setup using RocketPy  
- PID‑based thrust vector control loop  
- Neural network bias correction layer (digital twin augmentation)  
- Real‑time animated visualization of rocket attitude, thrust, and flame  
- Informative overlays showing thrust, angle, and simulation time
